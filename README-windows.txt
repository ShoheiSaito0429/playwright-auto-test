========================================
  Playwright Auto Test - Windows版
========================================

【必要なもの】
  Node.js (LTS版) → https://nodejs.org からダウンロード

【使い方】
  1. start.bat をダブルクリック
  2. 初回のみ: npm install と Playwright ブラウザの
     ダウンロードが自動で走ります（数分かかります）
  3. ブラウザに http://localhost:3200 が自動で開きます

【ツールの使い方】
  [記録モード]
  1. 「記録開始」タブで テスト対象のURLを入力
  2. 「記録開始」ボタンを押すとブラウザが起動します
  3. ブラウザ上で手動でログイン・操作を行ってください
  4. フィールドが自動検出されます
  5. テストケースタブで各フィールドに値を入力
  6. 「記録停止・保存」ボタンで保存

  [再生モード]
  1. 「再生」タブを開く
  2. 保存した記録とテストケースを選択
  3. 「テスト実行」ボタンで自動テストが走ります

【停止方法】
  start.bat のウィンドウを閉じる（または Ctrl+C）
